## 3. 无重复字符的最长子串

[leetcode](https://leetcode-cn.com/problems/longest-substring-without-repeating-characters/)

给定一个字符串 s ，请你找出其中不含有重复字符的 最长子串 的长度。

```
: s = "abcabcbb"
输出: 3 
解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。
```

思路：私用滑动窗口

先存入窗口，然后出现重复出现了就开始缩小窗口，最后取窗口的最大值

```cpp
class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        unordered_map<char,int> window;
        int left = 0,right = 0;
        int ans = 0;
        while(right < s.size()) {
            char c = s[right++];
            window[c]++;
            //如果出现重复了
            while(window[c] > 1) {
                char d = s[left++];
                window[d]--;
            }
            //记录滑动窗口大小
            ans = max(ans,right - left);
        }
        return ans;
    }
};
```

## 20. 有效的括号

[leetcode题目](https://leetcode-cn.com/problems/valid-parentheses/)

给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串 s ，判断字符串是否有效。

有效字符串需满足：

左括号必须用相同类型的右括号闭合。
左括号必须以正确的顺序闭合。

```
输入：s = "()"
输出：true

输入：s = "()[]{}"
输出：true
```

思路：

用一个哈希表存括号的反向

如果是左括号，直接进栈

如果是右括号，且匹配，就出栈

如果是有括号，且不陪陪，就 return false

最后栈空才返回 true


```cpp
class Solution {
public:
    bool isValid(string s) {
        unordered_map<char,char> mp;
        mp['}'] = '{';
        mp[')']  = '(';
        mp[']'] = '[';
        stack<char> st;
        for(int i=0;i<s.size();++i) {
            //如果是左括号
            if(mp.count(s[i]) == 0) { //找不到
                st.push(s[i]);
            }
            //右括号，且匹配
            else if(!st.empty() && st.top() == mp[s[i]]) {
                st.pop();
            }
            //右括号，不匹配
            else {
                return false;
            }
        }
        return st.empty() ? true : false;
    }
};
```

## 415. 字符串相加

[leetcode](https://leetcode-cn.com/problems/add-strings/)

给定两个字符串形式的非负整数 num1 和 num2 ，计算它们的和。

题解：

判断并求每个数的每一位，加上 sum，最后转字符 + ans

如果 sum 进 1 还得加上 sum

```cpp
class Solution {
public:
    string addStrings(string num1, string num2) {
        int index1 = num1.size() - 1;
        int index2 = num2.size()  - 1;
        string ans = "";
        int sum = 0;
        while(index1 >=0 || index2 >= 0 ) {
            int a = 0;
            int b = 0;
            if(index1 >= 0) {
                a = num1[index1--] - '0';
            }
            if(index2 >= 0) {
                b = num2[index2--] - '0';
            }
            sum += (a + b);
            int ge = sum % 10;
            int shi = sum / 10;
            ans = char(ge + '0') + ans;
            sum = shi;
        }
        if(sum > 0) {
            ans = char(sum + '0') + ans;
        }
        return ans;
    }
};
```
